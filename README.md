# 🎓 Smart Student Learning & Placement Management System

## 📌 Project Overview

This project is a full-stack web application built using the MERN stack (MongoDB, Express.js, React.js, Node.js). It provides a centralized platform for managing student learning, course tracking, assignments, and job placements in a training institute.



## 🚀 Features

### 👨‍💼 Admin

* Manage users (students, trainers)
* Manage courses
* Post job opportunities

### 👨‍🏫 Trainer

* Create and manage courses
* Create assignments
* Evaluate student submissions

### 👨‍🎓 Student

* Enroll in courses
* View and submit assignments
* Apply for jobs

---

## 🛠️ Tech Stack

* **Frontend:** React.js (Hooks, Router)
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Authentication:** JWT (JSON Web Token)
* **API Testing:** Postman

## ⚙️ Installation & Setup

### 🔹 Backend Setup

```
cd backend
npm install
npm run dev
```

### 🔹 Frontend Setup

```
cd frontend
npm install
npm start
```

## 🔐 Authentication & Authorization

* JWT-based authentication system
* Role-based access control:

  * Admin
  * Trainer
  * Student



## 📊 Modules Implemented

* ✅ User Authentication (Login/Register)

##  Objective

To build a centralized system where students can learn, track progress, and apply for jobs while trainers and admins efficiently manage educational and placement activities.
